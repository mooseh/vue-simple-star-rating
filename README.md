# vue-simple-star-rating

A Simple star rating component for Vue3

# installation
`npm -i vue-simple-star-rating`

# usage

## ES5
```
    require('vue-simple-star-rating')
```
## ES6
```
    import { StarRating } from 'vue-simple-star-rating'
```
then ...

```
    Vue.component(vueStarRating)

    or components: {
        StarRating
    }

```
